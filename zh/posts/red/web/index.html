css#

记录#

css的结构如下

有三种方式

  • inline CSS
  • internal CSS
    在head里,<style> …..</style>
  • external CSS
    需要在head里指定
    <link href=“style.css” rel=“stylesheet”/>

可以指定的属性

  • color
  • font-size(such as 24px)
  • font-family[serif,sans-serif]
  • text-transform[capitalize,lowercase,none,uppercase,inherit,initial,unset]
  • font-style[italic,normal,oblique,inherit,initial,unset]

Oblique characters follow the structure of the upright styles, while italics have a different structure, informed by cursive writing

  • line-height: 1.5 (行高是字体的1.5倍)
  • text-align[center,end,justify,left,right,start,inherit,initial,unset]

对于选择器有相同属性的,可以创建一个选择器列表

/* 与链接有关的伪类 */

text-decoration 可以将链接的下划线去掉
The CSS BOX MODEL

  • content: text,images,etc
  • border: a line around the element,still inside of the element
  • padding: invisible space around the content, inside of the element
  • margin: space outside of the element,between elements
  • fill area: area that gets filled with background color or background image


Final element width = left border + left padding + width + right padding + right border

介绍#

  • clear

    作用是清除元素的浮动问题

    • none 元素不会被向下移动以清除浮动
    • left 元素向下移动以清除右浮动
    • right 元素向下移动以清除左浮动
    • both 元素向下移动以清除左右浮动

  • 行级内容

    在CSS中，参与行内布局的内容被称为行级内容（inline-levelcontent）。默认情况下，大多数文本、替换元素以及生成的内容都是行级的。
    在行内布局中，通常将文本、替换元素以及其它的行级盒分段为一堆按顺序排列的盒子来进行布局。在每个行盒中，行级盒子相互垂直或者水平对齐，具体取决于书写模式。通常，它们与文本的基线进行对齐。这可以使用 CSS 进行更改。

  • 透明度

    • opacity: 背景上的文字的透明度也一起改变
    • rgba: RGB+ALPHA, rgba(xx,xx,xx,0,2)
