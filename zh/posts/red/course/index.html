课本#

A language isn’t something you learn so much as something you join.

语言是工具,思想是利器.计算机很强大,处理数据很快,但是同时它也很古板.“Programming is about a person using their real insight to build something useful,constructed out of these teeny,simple little operations that the computer can do"将计算机能够处理的微小东西组织起来.
在编程中,我们只会处理两种元素:函数和数据
导入函数库,python将已知函数和其他东西组织起来放入到模块里,这些模块共同组成了python库 e.g. math 模块提供了各种熟悉的数学函数 import 语句指定了需要模块的名称. 

>> from math import sqrt
>> sqrt(256)
16.0

使用赋值语句建立新的绑定
= 左边是名称,右边是值
名词也可以通过 import 语句绑定 e.g. 

from math import pi
pi * 2
...

将名称与值绑定，之后通过名称检索可能的值，就意味着解释器必须维护某种内存来记录名称、值和绑定，这种内存就是环境environment
名称也可以与函数绑定 

>>> f = max
>>> f
<built-in function max>
>>> f(2, 3, 4)
4

纯函数(Pure functions）：函数有一些输入(参数)并返回一些输出(调用返回结果) 

>>> abs(-2)
2

纯函数在调用时除了返回值外不会造成其他的任何影响,而且在使用相同的参数调用纯函数时总会返回相同的值
非纯函数(Non-pure functions):除了返回值外，调用一个非纯函数还会产生其他改变解释器和计算机的状态的副作用(side effect).一个常见的副作用就是使用print函数产生（非返回值的）额外输出。 

>>> print(1, 2, 3)
1 2 3

虽然print和abs在这些例子中看起来很相似，但它们的工作方式基本不同.print返回的值始终为None,这是一个不代表任何内容的特殊Python值.而交互式Python 解释器并不会自动打印None值，所以print函数的额外输出就是它的副作用.小心使用print函数,因为它返回None意味着它不适合用去赋值语句
