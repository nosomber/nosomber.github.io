快捷键#

从光标所在处删除到行首#

There is C-0 C-k or C-u 0 C-k
(setq debug-on-error t)

follow links without using mouse#

C c C o
或者设置 

;; RETURN will follow links in org-mode files
(setq org-return-follows-link  t)

基本使用#

Dired模式#

  • 修改文件名

    C x C q (wdired-change-to-wdired-mode) 然后 C x C s 保存

书签#

C x r m 设置标签
C x r b 跳转到书签
C x r l 列出书签

register 跳转#

类似于书签，但是重启 emacs 会丢失，特点是可以精细保存到光标所在位
C x r space 设置
C x r j 跳转

以只读方式打开文件#

C x C r 新的窗口打开
C x 4 r 当前窗口打开

批量替换#

在dired模式下,可以标记要查找替换的文件,然后执行 dired-do-query-replace-regexp
输入被替换的文字和替换文字,确认的时候有以下情况

  • y 替换当前
  • !替换所有
  • n忽略当前
  • C-g取消操作

窗口管理#

C x 2 类似划分窗口，同时打开当前的文件，向下分身
C x 3 向右边分身
C x 4 f 打开新的窗口 (find-file-other-window) 关于打开的窗口是水平还是垂直划分的，参考这里1
C x 4 C o 在新的窗口打开文件，但光标还在当前文件
C x 4 b 同上，但光标位置切换到新开的文件上
C x 1 删除其他窗口
恢复窗口布局，类似于 undo 和 redo 在窗口上的应用
使用一个包 winner-mode 

(use-package winner-mode
  :straight nil
  :hook (after-init . winner-mode))

移动#

M g g 移动到指定行
M v 向上滚动
C v 向下滚动
C M v 其他buffer向上滚动
C M S v 其他buffer向下滚动
C s 向下搜索
C r 向上搜索

文本操作#

  • 缩写替换

    C-x a i g (mnemonic add inverse global) and enter the expansion 記憶術 (Mnemonic forms)
    abbrev 的介绍
    输入 hw 后，使用 C-x a ' 会扩展为 hello world
    另外 M / 可以根据 buffer 里已有的补全

  • 矩形编辑

    C x SPC 将 range 转换成矩形 & 一矩形方式选择
    但是若想编辑，即整体添加，需要 C x r t 替换文本

org#

  • heading

    C RET 向下插入同级 heading
    M RET 向上插入…
    M 和方向键 调整层级大小
    M S 方向键 连带子集一起调节
    M 上下键 移动 heading

    • 有序列表 1. or 1)
    • 无须列表 - or +

    快速插入块 C c C ,
    e.g.

    • src
    • quote
    • example

  • org里的搜索

    C c / 可以根据不同的属性进行搜索

  • 将列表转换成 heading

    C c *
    对列表排序是 C c ^

    • he is
    • i want to
    • you

  • Todo

    将 heading 转换成 TODO 和切换 TODO 的状态可以用 C c C t 和 S 方向键
    快速插入一个 TODO M S RET
    TODO 状态关键字可以自定义

    • 全局设定(在配置文件里添加以下代码)用竖线分隔已完成状态和未完成状态 
      (setq org-todo-keywords
        '((sequence "TODO" "FEEDBACK" "VERIFY" "|" "DONE" "DELEGATED")))
    • 单独文件设定
      在文件头添加这样的属性,同样用 | 隔开 
      #+TODO: TOPLAY PLAYING | CLEAR NEWGANEPLUS

    设定关键字的样式 

    (setq org-tode-keyword-faces
      '(("TODO" . "#ff9977")
  ("DONE" . "#44bc44")))

    优先级
    C c , 设置优先级
    S 方向键 调整
    子任务
    在主任务后添加 [],在方括号里可以用 / 或者 % 两种方式表示进度

org#

  • 导出org时没有换行

    org中每空一行表示一个段落,要想在段落里换行有以下方法

    • 在文件头部加上
      #+OPTIONS: \n:t
    • 或者在配置文件加上 
    (setq org-export-preserve-breaks t)
    • 在换行处加上 \\

  • 刚打开是默认显示的标题级数

    两种方法

    • 在配置文件加上钩子(直接copy过来的,我没有定义快捷键,默认开启) 
      (defun org-show-two-levels ()
  (interactive)
  (org-content 2))

(add-hook 'org-mode-hook
          (lambda ()
            (define-key org-mode-map "\C-cm" 'org-show-two-levels)))
    • 在 org 头上可以加入下列指定语句(不过要注意,配置文件的优先级要高于此)
      • #+STARTUP show2levels (貌似可以指定到五级)
      • #+STARTUP content (以目录形式显示,即所有标题)
      • 还有 overview 和 showeverything 但每试过有啥用

normal#

  • 输入法

    使用 emacs-rime 作为输入法

    共享資料夾,包含預設輸入方案的源文件。 這些文件屬於 Rime 所發行軟件的一部份，在訪問權限控制較嚴格的系統上對用戶是只讀的，因此謝絕軟件版本更新以外的任何修改—— 一旦用戶修改這裏的文件，很可能影響後續的軟件升級或在升級時丟失數據。在 emacs 里是在 ~/.config/ibus/rime/
    用戶資料夾,則包含爲用戶準備的內容 将配置内容放在这里面就对了

    对于同步 在 installation.yaml 里，添加 installation_id: “linux-rime” 和 sync_dir: “home/mi/OneDrive/rime”
    okay, enjoy 😊

  • 重新加载此页面

    一般当外界文件变动的时候emacs不能很好地实时变动
    需要用到 revert-buffer 这个函数
    每次执行的时候还需要确认,sf上有人提供了修改版的 

    ;; Source: https://www.emacswiki.org/emacs/misc-cmds.el
(defun revert-buffer-no-confirm ()
    "Revert buffer without confirmation."
    (interactive)
    (revert-buffer :ignore-auto :noconfirm))

实践

Time:9/24/2023
扩大字体 setfont ter-122b (132b)

  1. 确保目前是 UEFI 模式 

    ls /sys/firmware/efi/efivars

  2. 连接网络
    before this *禁用 reflector*(可选)

    systemctl stop reflector.service

    这里直接略过，有线网络可以直接使用。无线网络需要配置 

    iwctl                           # 用这个工具配置
device list                     # 列出设备名，比如无线网卡看到叫 wlan0
station wlan0 scan              # 扫描网络
station wlan0 get-networks      # 列出网络 比如想连接YOUR-WIRELESS-NAME这个无线
station wlan0 connect YOUR-WIRELESS-NAME # 进行连接 输入密码即可
exit

  3. 硬盘分区

    • 先将硬盘转换成 gpt 类型 （不过大部分的硬盘已经是 gpt 类型的了）
      可忽略
      利用“fdisk -l”命令，若显示结果中的“Disk label
      type”的值为dos则分区是mbr，若该结果的值为gpt则分区是gpt
      (还未验证其可行性) 

      fdisk -l                   # 查看电脑磁盘的划分情况及盘符号
cfdisk 你自己电脑的盘符号     # 使用 cfdisk 对磁盘进行分区

    • 接下来用 cfdisk 工具对目标磁盘进行分区 boot & root & swap

      • boot 分区 type efi 550M
      • root 分区 type linux filesystem
      • swap 分区 type swap 4G(可以多但必要不大)

    • 分好区的磁盘进行格式化，以及开启 swap 

      mkfs.fat -F32 /dev/nvme0n1p1   #格式化 boot 分区
mkfs.ext4 /dev/nvme0n1p2       #格式化 root/home 分区

 # 这是格式化和开启交换分区的，如果前面没有创建交换分区，可忽略
mkswap /dev/nvme0n1p3       
swapon /dev/nvme0n1p3

  4. 挂载
    先挂载根分区，再挂载 EFI 分区 

    mount /dev/sdax（盘符号根据自己电脑而定）  /mnt  #挂载根目录
mkdir /mnt/boot     #创建boot目录（在根目录下）
# 挂载 boot 分区
mount /dev/sdax /mnt/boot

  5. 镜像源

    处于这个文件当中

    /etc/pacman.d/mirrorlist

    中国源 –放在最上方即可 

    Server = https://mirrors.ustc.edu.cn/archlinux/$repo/os/$arch             # 中科大
Server = https://mirrors.tuna.tsinghua.edu.cn/archlinux/$repo/os/$arch    # 清华大学

    一些国际源 

    Server = https://mirror.archlinux.tw/ArchLinux/$repo/os/$arch   #东亚地区:中华民国
Server = https://mirror.0xem.ma/arch/$repo/os/$arch    #北美洲地区:加拿大
Server = https://mirror.aktkn.sg/archlinux/$repo/os/$arch    #东南亚地区:新加坡
Server = https://archlinux.uk.mirror.allworldit.com/archlinux/$repo/os/$arch    #欧洲地区:英国
Server = https://mirrors.cat.net/archlinux/$repo/os/$arch    #东亚地区:日本

  6. 必须的包
    > pacman -Sy archlinux-keyring # 安装这个可解决证书问题 

    pacstrap /mnt base base-devel linux linux-headers linux-firmware  #base-devel在AUR包的安装是必须的

    一些工具性的包 

    pacstrap /mnt vim bash-completion   # vim 和 bash 的补全
#单写出来 networkmanager 需要安装，不然安装完成后，会连接不上网络（当然需要开启这个服务，稍后再说。）
pacstrap /mnt networkmanager

  7. 生成fstab文件

    fstab 用来定义磁盘分区 

    genfstab -U /mnt >> /mnt/etc/fstab

    复查一下 /mnt/etc/fstab 确保没有错误 不必要 

    cat /mnt/etc/fstab

  8. Change root

    把环境切换到新系统的/mnt 下 

    arch-chroot /mnt

    设置时区，在/etc/localtime 下用/usr
    中合适的时区创建符号连接。如下设置上海时区。 

    ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

    随后进行硬件时间设置，将当前的正确 UTC 时间写入硬件时间。 

    hwclock --systohc

  9. 设置Locale进行本地化

    编辑 /etc/locale.gen

    • 去掉 en_US.UTF-8 所在行
    • 以及 zh_CN.UTF-8 所在行的注释符号（#）。

    然后使用如下命令生成 locale。 

    locale-gen

    最后向 /etc/locale.conf 导入内容 

    echo 'LANG=en_US.UTF-8'  > /etc/locale.conf

  10. 设置主机名

    首先在=/etc/hostname=设置主机名 

    vim /etc/hostname

    加入你想为主机取的主机名，这里比如叫 apple

    接下来在=/etc/hosts=设置与其匹配的条目。 

    vim /etc/hosts

    加入如下内容 

    127.0.0.1   localhost
::1         localhost
127.0.1.1   apple（刚才设置的主机名）

  11. 为root设置密码 

    passwd root

  12. 制作Grub引导 

    pacman -S grub efibootmgr amd-ucode    # 如果是intel就装intel-ucode

mkdir /boot/grub
grub-install --target=x86_64-efi --efi-directory=/boot
grub-mkconfig > /boot/grub/grub.cfg

事后可设置 /etc/default/grub 中 可提升启动速度
GRUB_CMDLINE_LINUX_DEFAULT="loglevel=3 nowatchdog"

  13. 完成 

    exit                # 退回安装环境#
umount -R  /mnt     # 卸载新分区
reboot              # 重启

    静音和调大调小之间的不协调
    首先是在调节音量之前判断一下系统现在是否处在静音的状态.
    如果是的话，那么将其解除静音.

    代码实现如下 

    #!/bin/sh
# NAME
#     volume.sh - Increase/decrease/mute volume using PulseAudio pactl
# SYNOPSIS 
#     volume.sh <raise|lower|mute>
# Suggested location: ~/scripts/volume.sh

step=2   # number of percentage points to increase/decrease volume
max=50  # do not increase volume above this percentage

# Prints current volume percentage, e.g. `60`, `95`, `20`, etc.
function get_current_volume() {
  pactl get-sink-volume @DEFAULT_SINK@ | awk -F '/' '{print $2}' | grep -o '[0-9]\+'
}

mute_or_not=`pactl get-sink-mute @DEFAULT_SINK@ | awk '{print $2}'`  # 查看是否静音

if [ $1 == "raise" ]; then
  # Modification: only increase volume if current volume is less than `max`
  if [ `get_current_volume` -lt ${max} ]; then
      if [ $mute_or_not == "yes" ]; then 
        pactl set-sink-mute @DEFAULT_SINK@ toggle
      fi
        pactl set-sink-volume @DEFAULT_SINK@ "+${step}%"

  fi

# The rest of the script is identical to the original `volume.sh`
elif [ $1 == "lower" ]; then
    if [ $mute_or_not == "yes" ]; then 
        pactl set-sink-mute @DEFAULT_SINK@ toggle
    fi
    pactl set-sink-volume @DEFAULT_SINK@ "-${step}%"

elif [ $1 == "mute" ]; then
    pactl set-sink-mute @DEFAULT_SINK@ toggle
else
    echo "Unrecognized parameter: ${1}"
    echo "Usage should be: volume.sh <raise|lower|mute>"
  fi

  • 给 Arch 添加 Swap 文件

    参考：TimothyQiu’s Blog

    起因：之前按照 cc 和 archstory 的步骤，给 arch 开机的时候发现 arch
    显示/找不到/swapfile/，因此在网上有找了一些其他的方法（感觉 cc
    的有些省略） 📍创建 Swap 文件

    创建 Swap 文件有很多种方法，但最可移植的方法是使用 dd=。比如创建 1G 的 =/swapfile ： 

    dd if=/dev/zero of=/swapfile bs=1M count=1024 status=progress
chmod 0600 /swapfile
mkswap -U clear /swapfile

    这里的 chmod 是为了安全，所有人都可读写的 Swap 文件是巨大隐患。
    mkswap 的 -U 参数用于设置 swap 的 UUID。但是因为 swap
    文件必须使用文件系统路径去指定，所以这里使用特殊的 clear
    作为参数去清空它（实际效果是设置了全零的 UUID）。

    📍打开与关闭

    创建好 swap 文件后，就可以直接启用了： 

    swapon /swapfile

    最后把它放进 /etc/fstab 里，这样每次启动就会直接启用： 

    /swapfile none swap defaults 0 0

    如果是要关闭，顾名思义就： 

    swapoff /swapfile

    关闭以后它就是个普通文件，想删除就可以直接删除了。

    📍使用 Swap 文件进行休眠

    需要设置=resume=和=resume_offset=参数

    resume 参数是 swap 文件所在的分区（例如 =/dev/nvme0n1p2=，或者
    =UUID=4209c845-f495-4c43-8a03-5363dd433153=）有两种表示方法
    =UUID=的获取方法是 

    findmnt -no UUID -T /swapfile

    =resume_offset=获取方法 

    filefrag -v /swapfile | awk '{ if($1=="0:"){print substr($4, 1, length($4)-2)} }'

    编辑=/etc/default/grub=，将以上两个参数写进=/etc/default/grub=中
    像这样 

    GRUB_CMDLINE_LINUX_DEFAULT="loglevel=3 nowatchdog resume=UUID=db7c6671-e10c-4486-9077-87692688cdcc resume_offset=3733504"

    最后重新生成一下`=grub.cfg= 

    grub-mkconfig -o /boot/grub/grub.cfg

    📍initramfs

    如果你的 initramfs 没有使用 systemd 钩子（使用的是
    base），那么就还需要添加一个 resume 钩子才会尝试从休眠中恢复：

    编辑 /etc/mkinitcpio.conf 文件，在 HOOKS=(...) 里的 udev
    之后的任何位置加入 =resume=。最后重新生成 initramfs 即可： 

    mkinitcpio -p linux

  • 和 Win 时间不同

    原因

    少数派的一篇文章

    GMT: Greenwhich Mean Time
    (格林尼治标准时间)，也就是世界时间。以地球自转为基础的时间计量系统。但是GMT不够准确（相比较而言）

    *UTC*：Universal Time Coordinated
    (协调世界时)，以=原子时=的秒长为基础。现在作为时间标准时间使用

    *RTC*：Real-Time Clock 实时时钟 ，在计算机领域称为硬件时间

    e.g 北京位于东八区， 其时间即为 UTC+8

    Windows把硬件时钟设为=本地时间= Linu把硬件时间设为=UTC=

    那么Linux显示的时间就是UTC时间➕时区

    解决方法

    1. Windows

      管理员打开=powershell= 

      Reg add HKLM\SYSTEM\CurrentControlSet\Control\TimeZoneInformation /v RealTimeIsUniversal /t REG_DWORD /d 1

      或者打开「注册表编辑器」，定位到
      计算机\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\TimeZoneInformation
      目录下，新建一个 DWORD 类型，名称为 RealTimeIsUniversal
      的键，并修改键值为 1 即可。

    2. Linux 

      timedatectl set-local-rtc 1 --adjust-system-clock

    If ‘split-height-threshold’ specifies an integer, WINDOW is at least ‘split-height-threshold’ lines tall and can be split vertically, split WINDOW into two windows one above the other and return the lower window. Otherwise, if ‘split-width-threshold’ specifies an integer, WINDOW is at least ‘split-width-threshold’ columns wide and can be split horizontally, split WINDOW into two windows side by side and return the window on the right. If that succeeds, return the lower window. Return nil otherwise.

Alternatively you could set split-height-threshold to nil, preventing it from ever splitting vertically, or set the variable split-window-preferred-function to a different function.
