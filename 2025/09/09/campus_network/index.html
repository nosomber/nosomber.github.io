使用nmcli连接WPA2 802.1X的无线网
2025-09-09 422字

连接需要用户名和密码的Wi-Fi，IEEE 802.1x
使用nmcli当作普通的只需要密码的来连接时会报错 Failed to determine AP security information
新建一个连接名称 NAME

  • nmcli device status 确定无线网卡名称 DEVICE
  • nmcli device wifi list 扫描Wi-Fi，确定 SSID

下面是格式，大写的位置需要替换成自己的

nmcli connection add type wifi con-name NAME ifname DEVICE ssid SSID \
    -- wifi-sec.key-mgmt wpa-eap \
    802-1x.eap ttls \
    802-1x.phase2-auth mschapv2 \
    802-1x.identity USERNAME \
    802-1x.password PASSWORD

然后就可以连接 NAME 这个别名
nmcli con up NAME
