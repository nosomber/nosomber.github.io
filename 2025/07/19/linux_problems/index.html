1. DONE There is no trackikmng information for the current branch. Please specify which branch you want to merge with.

这个错误通常出现在你创建了新的分支并推送到远程仓库，但没有设置该分支与远程分支之间的关联，导致 Git 无法知道你想要和哪个远程分支进行合并

  • 将把本地 main 分支与远程的 origin/main 分支关联起来
    git branch --set-upstream-to=origin/main main
  • 将远程的 main 分支合并到当前分支
    git pull origin main

2. DONE ssh: connect to host github.com port 22: Connection timed out fatal: Could not read from remote repository.

使用 SSH 的话，GitHub 支持通过端口 443 建立 SSH：
编辑 SSH 配置文件：

vim ~/.ssh/config

Host github.com
  HostName ssh.github.com
  Port 443
  User git

3. DONE github仓库已经有内容了，如果我还想将本地的一个文件夹提交到那个仓库（覆盖）

git remote add origin git@github.com:nosomber/nosomber.github.io.git
git push -u origin main --force

# git 切换到 main 分支
git branch -M main

4. DONE dial tcp: lookup aur.archlinux.org: no such host

无法解析域名 aur.archlinux.org，说明 DNS 查询失败

#编辑 /etc/resolv.conf
nameserver 1.1.1.1     # Cloudflare DNS
nameserver 8.8.8.8     # Google DNS

5. DONE Failed to remove device: org.bluez.Error.NotReady

重启蓝牙配适器
sudo rfkill block bluetooth
sudo rfkill unblock bluetooth
或重启蓝牙服务
sudo systemctl restart bluetooth

6. DONE Failed to connect: org.bluez.Error.Failed br-connection-page-timeout

连接超时，尝试重启蓝牙服务
sudo systemctl restart bluetooth

7. DONE windows和linux双系统蓝牙设备

蓝牙配对密钥（link key） 在 Windows 和 Linux 之间没有同步，耳机记住了其中一个系统里的密钥，但和本系统里面的不一样。需要先 忘记设备，重新连接

bluetoothctl
remove xxx
scan on
connect xxx
exit
