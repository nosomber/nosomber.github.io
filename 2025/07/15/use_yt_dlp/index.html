使用yt-dlp
2025-07-15 703字

Mac用户从 链接 下载程序

通过终端找到文件下载的位置 

ls         #列出当前文件夹下的文件和文件夹
cd xxx     #进入 xxx 这个文件夹

#应该可以看到有 yt-dlp_macos 这个文件
sudo chmod +x yt-dlp_macos   #为这个程序添加可执行权限
#然后就可以运行这个程序

由于 Safai 使用的是 Apple 自家的加密机制（Keychain + SQLite），cookie 存储格式不公开。 所以推荐使用 Google Chrome 来进行下面步骤 使用 Chrome 打开 youtube 网站后 

./yt-dlp --cookies-from-browser chrome "https://example.com/video"

这样就可以下载指定链接的视频内容了
